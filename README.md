# Simple Image upload app
Simple web app for image upload and preview.

## Installation

### Steps

1. Clone the repo
  ```bash
    git clone https://github.com/flixusk/Simple-image-upload-app.git
  ```
2. CD to frontend
  ```bash
    cd frontend
  ```
3. Install pakages
  ```bash
    npm install
  ```
4. Run app
  ```bash
    npm run dev
  ```
